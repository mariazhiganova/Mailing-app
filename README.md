# Проект "MAILFLOW"

## Описание

В проекте реализован сервис управления email-рассылками. Приложение разработано на Django с
системой разделения прав доступа и кешированием для повышения производительности.

### Описание модулей

В пакете **config** реализован проект и необходимые конфигурационные файлы.

В пакете **mail_flow** расположено основное приложение для управления рассылками:

- Модели для рассылок, сообщений, получателей и попыток отправки
- Контроллеры с системой разделения прав доступа
- Сервисы для отправки рассылок и сбора статистики
- Миксины для контроля доступа пользователей и менеджеров
- Настроено кеширование страниц со списками рассылок, получателей и отчетов

В пакете **users** реализовано управление пользователями:

- Кастомная модель пользователя с дополнительными полями
- Контроллеры для регистрации, аутентификации и управления профилем
- Система разрешений для разделения функционала между пользователями и менеджерами
- Возможность загрузки аватара профиля

В директории **templates** находятся HTML шаблоны для всех страниц приложения.

В директории **static** расположены статические файлы: стили, изображения и скрипты.

В пакете **management/commands** содержатся кастомные команды для управления рассылками.

### Функциональность

#### Для пользователей:

- Создание, просмотр, редактирование и удаление своих рассылок, сообщений и получателей
- Просмотр статистики по своим рассылкам
- Управление личным профилем с возможностью загрузки аватара

#### Для менеджеров:

- Просмотр всех рассылок и получателей в системе
- Просмотр списка пользователей сервиса
- Блокировка пользователей сервиса
- Отключение рассылок

### Система кеширования

В проекте реализовано серверное кеширование:

- Кеширование страниц со списками на 15 минут
- Низкоуровневое кеширование данных статистики
